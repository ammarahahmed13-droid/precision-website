## Typography

### Font Family

**Inter** (Google Fonts)

### Type Scale

| Element | Size (Desktop) | Weight | Color |
|---------|---------------|--------|-------|
| H2 Heading | 72px | 700 | `#FFFFFF` |
| H3 Panel Title | 36px | 700 | `#FFFFFF` |
| H4 Card Title | 18px | 700 | `#FFFFFF` |
| Section Label | 13px | 600 | `#00BFFF` |
| Body Large | 20–21px | 400 | `#D1D5DC` |
| Body | 18px | 400 | `#D1D5DC` |
| Body Small | 15px | 500 | `#D1D5DC` |
| Caption / Meta | 13px | 600 | `#9CA3AF` |
| Badge | 12px | 700 | Accent |
| Fine Print | 14px | 400 | `#9CA3AF` |

### Responsive Scaling

| Element | Mobile | Tablet | Desktop |
|---------|--------|--------|---------|
| H2 | 40px | 64px | 72px |
| Body Large | 18px | 20px | 21px |
| Panel Title | 28px | 36px | 36px |

### Style Notes

- Line height: 1.1 headings, 1.45 body
- Letter spacing: `0.08em–0.12em` on uppercase labels/badges
- Gradient text: `background-clip: text` with brand gradient
- `-webkit-text-fill-color` overrides on headings for Bricks compatibility
